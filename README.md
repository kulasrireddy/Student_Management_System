# 🎓 Student Management System

A modern and responsive Student Management System built using Flask and SQLite with secure authentication and CRUD operations.

---

## Features

User Registration & Login  
Secure Password Hashing  
Add Students  
Edit Students  
Delete Students  
Search Students  
Responsive Dashboard  
Modern Colorful UI  
Flash Messages  
SQLite Database Integration  
Dashboard Statistics Cards  

---

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Installation

### 1.Clone Repository

```bash
git clone https://github.com/yourusername/student-management-system.git
```

### 2. Open Project Folder

```bash
cd student-management-system
```

### 3. Install Requirements

```bash
pip install -r requirements.txt
```

### 4. Run Application

```bash
python app.py
```
